This project compilation and running is dependent on Apache-Ant. If it is not present in your OS please follow instructions from
http://ant.apache.org/manual/install.html

1) Open command line and navigate to this project directory.
2) You should be able to see "build.xml" when you type "ls" command after Step 1.
3) Type "ant build-project" and hit enter to build the project. Type "clean-build-project" and hit enter 
to clean and build the project.
4) Type "ant SMCustomMatch" and hit enter to run the project.
